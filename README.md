# opentrainingcenter_spring
OpenTrainingCenter with Spring and Vaadin

Start it with 


```jshelllanguage
mvn spring-boot:run -Drun.jvmArguments="-Dspring.profiles.active=dev"
```
or 
```jshelllanguage
mvn spring-boot:run -Drun.jvmArguments="-Dspring.profiles.active=prod"
```
